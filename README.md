
# 🎉 Campus Event Management System (CEMS)

A full-stack web application that allows users to create, explore, and register for events with secure authentication.

---

## 🚀 Live Demo

- 🌐 Frontend: https://cems-six.vercel.app  
- ⚙️ Backend API: https://cems-e5eo.onrender.com  

---

## 🧠 Overview

CEMS is a full-stack application designed to manage campus events.  
It supports user authentication, event creation, and event registration using a relational database and REST APIs.

---

## ✨ Features

- 🔐 User Authentication (Signup/Login with JWT)
- 📅 Create Events (Authenticated users)
- 📖 View All Events
- 📝 Register for Events
- 👤 View Registered Events (My Events)
- 🚪 Logout functionality
- ⏳ Loading indicators & feedback messages
- 🌍 Deployed full-stack application

---
